# Flash Projects

A collection of Adobe Flash coursework projects created during undergraduate study.
Each mini-project includes its source file (`.fla`) and exported movie (`.swf`).

## Project Files

- `aquarium.fla`
- `auqarium.swf`
- `final tree.fla`
- `final tree.swf`
- `first.fla`
- `first.swf`
- `planet.fla`
- `planet.swf`
- `truck.fla`
- `truck.swf`

## Notes

- File names are kept as originally created.
- `auqarium.swf` uses the original spelling from the source folder.

## Requirements

To edit `.fla` files:
- Adobe Animate (or legacy Macromedia/Adobe Flash authoring tools)

To play `.swf` files:
- Adobe Flash Player Projector (standalone player), or
- A compatible SWF runtime/emulator (for example, Ruffle where supported)

## How To Use

1. Clone the repository.
2. Open any `.fla` file in Adobe Animate to inspect or modify the project.
3. Open any `.swf` file in a standalone Flash-compatible player to preview output.

## Repository Purpose

This repository is an archive of learning projects and examples made with Flash.
